

#Dictionary
Import list of word and translate of it to program and create binary search tree

Program menu
Show Binary tree
Translate word
Add word
Delete word
Edit word
Export to file


## Dependencies software
GNU Compiler Collection

##USAGE
###Compile
Run
```
make
```
###Run
```
./dict <input-file>
```
`<input-file>` a file which contain list of word and trans


File input
```
word.txt
```
